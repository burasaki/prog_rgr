# Шифрование текста в изображениях (BMP)

Консольное приложение на языке C для скрытия текстовых сообщений в BMP-файлах с использованием метода наименьшего значащего бита (LSB — Least Significant Bit).

##  Структура проекта
- [main.c](main.c) — точка входа.
- [script.c](script.c) — реализация функций стеганографии, логирования и чтения конфигурации.
- [RGR.h](RGR.h) — заголовочный файл с описанием структур (BITMAPFILEHEADER, BITMAPINFOHEADER, config_t) и прототипов функций.
- [tests/test.c](tests/test.c) — модульные тесты на CMocka.
- [CMakeLists.txt](CMakeLists.txt) — конфигурационный файл для сборки проекта.
- [config.txt](config.txt) - Конфигурация программы
- [images](images/) - примеры BMP изображений
## Конфигурация приложения
```
mode = 1
input_image = ../images/Car.bmp
encrypt_image = ../enc/Car.bmp
text_to_hide = Ваше секретное сообщение
use_red = 0
use_green = 1
use_blue = 1
```
- **mode**: режим работы приложения. 1 — зашифровать текст в картинку, 2 — дешифровать текст из картинки.

- **input_image**: путь к исходному изображению BMP.

- **encrypt_image**: путь к файлу, в который сохранится результат (при шифровании), либо из которого будет считываться текст (при дешифровании).

- **text_to_hide**: строка текста для скрытия (используется в режиме mode = 1).

- **use_red, use_green, use_blue**: флаги активации цветовых каналов (1 — включен, 0 — выключен). Текст будет распределяться/считываться только по выбранным каналам.

## Сборка
``` bash
mkdir build && cd build
cmake ..

```
